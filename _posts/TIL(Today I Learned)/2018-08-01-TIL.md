## 멘붕의 연속

- 키워즈: 객체, 전역변수, 지역변수, 값에 의한 호출, 참조에 의한 호출, git 사용법
- 오늘 초반엔 각자 코드 만들어보는 시간을 가졌는데, 2차원 배열만 내내 하다가 시간을 보냈다. 
- 전역변수는 코드 전체에 적용되는 변수이고 지역변수는 함수 내 존재하는 변수이다. 전역변수는 최소한으로 쓰는 것이 좋다.
```javascript
var arr = [1,2,3];
var paw = function(arr){
    arr.push(4);
    console.log(arr);
    arr = [1,2,3,4,5];
    console.log(arr);
};
arr = [1,2,3];
paw(arr) = [1,2,3,4]
```
