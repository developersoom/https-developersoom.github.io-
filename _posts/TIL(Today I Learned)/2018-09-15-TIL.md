## git 이전 커밋 수정

[출처](http://woodz.tistory.com/75)


1. git log에서 커밋명 확인 

2. 이전 커밋으로 이동 
```text
$ git rebase --interactive 2f61d921981a45c687e5dc26d6f7e6a3925ca0e5^

끝의 ^ 는 못보고 안했는데 잘됐다....  뭐지
```

3. 내가 원하는 커밋에 가서 'pick'을 'edit'으로 수정

4. 커밋내용 수정
```text
$ git commit --amend
```

5. rebase
```text
$ git rebase --continue
```


- 깃헙 페이지 테마 수정하다가 새벽 2시가 다되도록 못자고 있다..
