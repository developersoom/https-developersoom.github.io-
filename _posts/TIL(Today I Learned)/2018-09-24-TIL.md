(2) sort()로 문자열 정렬 가능 <br>

- 공부할 것(아래 함수 의미 파악하기)
    ```text
     lowerCases.sort(function (a, b) {
         if (a > b) {
             return -1;
         }
         if (b > a) {
             return 1;
         }
         return 0;
     });
    ```

### Compiling
- 컴퓨터는 이진법만 이해하기 때문에 우리가 쓴 소스코드(source code)는 컴퓨터가 실제로 사용하는 머신코드(machine code)인 0과 1로 변환되어야 한다.
- 이 변환은 컴파일러(compiler)라는 소프트웨어를 통해 이루어진다.
